# q2a-korean-translation

Question2Answer 한국어 번역

<https://github.com/minwook-shin/q2a-korean-translation> 에 있는 것을 포크했음. 추가 보완하려함.


## Available versions

* Q2A 1.8.x 번역하려함

## 시작

### 설치

1. php 파일 8개 내려받음
1.  ```ko``` 디렉토리를 ```qa_lang``` 디렉토리에 복사

여러분의 기여를 환영합니다.
